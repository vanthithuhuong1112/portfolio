# Thu Huong Portfolio (HTML + CSS)

## Cách chạy nhanh (offline)
1) Giải nén thư mục.
2) Mở file `index.html` bằng trình duyệt (double click).

## Để up GitHub Pages (miễn phí)
1) Tạo repo mới trên GitHub (ví dụ: `thu-huong-portfolio`).
2) Upload toàn bộ folder (index.html, styles.css, assets/...).
3) Repo → Settings → Pages → Source: `main` branch → `/ (root)` → Save.
4) Chờ 1-2 phút, GitHub sẽ cung cấp link website.

## File Single
- `thu-huong-portfolio-single.html`: 1 file duy nhất (CSS nằm trong file). 
- Nếu dùng file single, bạn vẫn có thể để CV PDF trong folder và chỉnh link nếu cần.
